# 岗位核心能力模型库 v2

评估简历时必须先匹配角色类型，再使用对应能力模型。禁止硬编码某一岗位的考察维度。

---

## 使用方式

1. 从意向岗位名 → 在能力库中匹配「名义模型」
2. 从 JD 的实际职责描述 → AI 判断实际角色类型
3. 如果名义 ≠ 实际（如岗位写产品运营但 JD 80% 是数据标注）→ 使用「实际模型」评估，并在报告中标注差异
4. 如果 JD 横跨多个子类型 → 混合模型，按 JD 权重分配
5. 如果角色类型不在库中 → 基于 JD 自行提取 3-4 个维度并注明「未命中模型库，由 AI 提取」

## 运营大类

### 产品运营

```
核心公式：产品运营 = 数据驱动 + 策略执行 + 效果验证 + 效率优化

考察维度：
1. 数据驱动 — 用 SQL/Excel 从数据中发现问题并指导运营决策
   证据：指标监控、异常定位、数据复盘报告
2. 策略执行力 — 把运营策略变成具体动作并落地
   证据：活动策划、内容供给优化、用户触达、激励方案
3. 效果验证力 — 用转化率/留存率等指标衡量运营效果
   证据：AB测试、漏斗分析、ROI核算
4. 效率优化力 — 通过 SOP/工具提升运营效率
   证据：SOP搭建、自动化脚本、人效提升数据

常见错误：用产品经理的「需求→方案→上线」逻辑写运营经历；数据只提取不分析
动作词库：推动、落地、沉淀、优化、搭建、监控、复盘、协同
```

### 用户运营

```
核心公式：用户运营 = 用户洞察 + 分层触达 + 留存策略 + 数据验证

考察维度：
1. 用户洞察力 — 理解用户行为动机，定位流失原因
   证据：用户画像、行为分析、流失归因、反馈收集
2. 分层触达力 — 对不同用户群体差异化运营
   证据：RFM分群、push策略、权益设计、召回方案
3. 留存策略力 — 通过策略减少用户流失
   证据：留存率提升、流失预警、VIP体系、社群活跃度
4. 数据验证力 — 策略效果可量化
   证据：留存率/活跃度/LTV/召回率变化

常见错误：把客服工作写成用户运营；用社群人数代替实际活跃数据
动作词库：运营、触达、留存、召回、分层、转化、活跃
```

### 内容运营

```
核心公式：内容运营 = 内容策略 + 供应链管理 + 质量把控 + 消费效果

考察维度：
1. 内容策略力 — 制定内容方向和节奏
   证据：内容规划、热点策划、品类矩阵、竞品对标
2. 供应链管理力 — 管理内容生产和供给稳定
   证据：创作者运营、征稿机制、内容采购、产能监控
3. 质量把控力 — 确保内容品质和合规
   证据：审核标准、质量评分、合规机制、内容去重
4. 消费效果力 — 内容对消费指标的实际影响
   证据：VV/CTR/互动率/消费时长变化

常见错误：用发布量代替消费数据；质量提升无客观标准
动作词库：策划、供给、审核、分发、优化、监控、汰换
```

### 活动运营

```
核心公式：活动运营 = 创意策划 + 执行落地 + 资源协调 + 效果复盘

考察维度：
1. 创意策划力 — 设计吸引人的活动玩法和传播策略
   证据：活动方案、玩法设计、权益规划、传播策略
2. 执行落地力 — 方案完整执行到位
   证据：排期管理、物料统筹、风险预案、执行SOP
3. 资源协调力 — 调动内外部资源支持活动
   证据：跨部门协调、预算管理、供应商管理、站内外联动
4. 效果复盘力 — 总结可复用的经验
   证据：ROI核算、漏斗分析、AB测试结论、复盘报告

常见错误：用曝光量代替转化数据；年度大促写成独立成果
动作词库：策划、统筹、执行、协调、优化、复盘、转化
```

### 商家运营 / ToB 运营

```
核心公式：商家运营 = 招募策略 + 培育赋能 + 分层管理 + GMV驱动

考察维度：
1. 招募策略力 — 找到并吸引优质商家入驻
   证据：招商方案、商家画像、入驻门槛、渠道拓展
2. 培育赋能力 — 帮助商家成长和提升经营能力
   证据：培训体系、经营指导、工具赋能、标杆打造
3. 分层管理力 — 对商家差异化运营
   证据：KA/腰部/尾部分层、权益设计、激励政策
4. GMV驱动力 — 商家运营对平台交易额的实际贡献
   证据：GMV变化、动销率、商家留存率、客单价变化

常见错误：把商家日常沟通维护写成商家运营；GMV增长无归因分析
动作词库：招募、培育、管理、赋能、提升、驱动、分层
```

### 电商运营

```
核心公式：电商运营 = 货品策略 + 流量获取 + 转化优化 + 复购留存

考察维度：
1. 货品策略力 — 选品、定价、库存管理和爆款打造
   证据：品类规划、SKU管理、价格策略、库存周转优化
2. 流量获取力 — 站内站外流量渠道的拓展和优化
   证据：搜索排名、推荐流量、广告投放、social引流
3. 转化优化力 — 优化商品详情、促销策略和购物链路
   证据：转化率变化、客单价变化、促销ROI
4. 复购留存力 — 通过用户运营提升复购和LTV
   证据：复购率、会员体系、老客召回、RFM分层

常见错误：GMV增长归因为大盘/促销而非自己的运营动作；活动描述无策略逻辑
动作词库：选品、定价、投放、促销、转化、复购、引流
```

### AI 产品运营

```
核心公式：AI产品运营 = 产品运营基础 + 模型理解 + 数据质量 + 合规风控
（本岗位需要兼备产品运营的全部能力，以下为 AI 特有维度）

考察维度：
1. 产品运营基础 — 同「产品运营」全部 4 个维度
2. 模型理解力 — 理解 AI 模型的能力边界、行为模式、常见失败模式
   证据：Prompt工程、模型评测、badcase分析、效果迭代报告
3. 数据质量力 — 保证训练数据和评测数据的质量
   证据：标注SOP搭建、标注质量管控、数据清洗、一致性验证
4. 合规风控力 — 管理 AI 产品的合规风险和内容安全
   证据：NSFW策略、合规审核流程、风险事件处理、用户投诉闭环

常见错误：把数据标注纯执行写成 AI 产品运营；模型效果归因为自己的 Prompt 优化
动作词库：评测、迭代、标注、合规、监控、闭环、搭建
```

### 增长运营

```
核心公式：增长运营 = 增长模型 + 渠道实验 + 转化优化 + 规模化

考察维度：
1. 增长模型力 — 构建和拆解增长漏斗，识别增长杠杆点
   证据：AARRR模型应用、增长实验设计、假设验证
2. 渠道实验力 — 多渠道获客和效果对比
   证据：投放优化、素材AB测试、渠道ROI分析、获客成本控制
3. 转化优化力 — 从获客到激活到付费的全链路优化
   证据：转化率提升、首单/首活策略、新用户引导
4. 规模化为 — 将有效的增长策略固化为可复用的机制
   证据：增长playbook、自动化获客系统、referral机制

常见错误：用自然增长数据冒充策略成果；获客成本无渠道级归因
动作词库：增长、获客、转化、实验、规模化、裂变、优化
```

### 商业化运营

```
核心公式：商业化运营 = 变现策略 + 定价优化 + 付费转化 + 收入增长

考察维度：
1. 变现策略力 — 设计付费产品组合和变现路径
   证据：SKU设计、权益分层、免费转付费策略
2. 定价优化力 — 通过数据和实验优化定价
   证据：价格弹性测试、套餐优化、折扣策略、竞品对标
3. 付费转化力 — 提升从免费到付费的转化效率
   证据：付费渗透率、首购转化、续费率、ARPU变化
4. 收入驱动力 — 运营动作对收入的直接贡献
   证据：收入增长、ARPU提升、流失挽回收入、ROI核算

常见错误：收入增长归因于大盘而非自己的策略；续费数据无基数
动作词库：变现、定价、付费、转化、增收、续费、渗透
```

### 策略运营

```
核心公式：策略运营 = 规则设计 + 数据建模 + 策略调优 + 效果验证

考察维度：
1. 规则设计力 — 设计运营策略的规则和逻辑
   证据：补贴规则、流量分配策略、定价算法、风控规则
2. 数据建模力 — 用数据分析支撑策略设计
   证据：用户分群模型、ROI预测、LTV预估、流失预测
3. 策略调优力 — 基于效果数据持续优化策略参数
   证据：AB测试、增量分析、策略迭代记录
4. 效果验证力 — 证明策略变化带来了增量效果
   证据：增量归因、对照组对比、长周期效果追踪

常见错误：策略效果无AB验证；把执行层面的操作写成策略设计
动作词库：设计、调优、验证、建模、归因、迭代、分配
```

### 新媒体运营

```
核心公式：新媒体运营 = 内容创作 + 平台玩法 + 粉丝增长 + 流量转化

考察维度：
1. 内容创作力 — 产出符合平台调性的内容
   证据：爆款内容数、内容矩阵、选题策划、脚本撰写
2. 平台理解力 — 熟悉各平台的算法和流量机制
   证据：平台数据分析、推荐机制理解、热点追踪
3. 粉丝增长力 — 通过内容驱动账号增长
   证据：粉丝增长曲线、互动率、转评赞数据
4. 流量转化力 — 将社媒流量转化为业务价值
   证据：导流数据、种草转化、品牌声量提升

常见错误：用单条爆款数据代表整体运营能力；粉丝数≠活跃粉
动作词库：创作、运营、涨粉、转化、种草、传播、互动
```

### 渠道运营

```
核心公式：渠道运营 = 渠道拓展 + 投放优化 + 商务谈判 + ROI分析

考察维度：
1. 渠道拓展力 — 发现和拓展有效的获客渠道
   证据：渠道矩阵、新渠道开拓、渠道关系维护
2. 投放优化力 — 通过数据和素材优化提升投放效率
   证据：CTR/CVR优化、素材AB测试、出价策略
3. 商务谈判力 — 争取更优的渠道合作条件
   证据：返点谈判、独家合作、资源置换
4. ROI分析力 — 精准核算每个渠道的投产比
   证据：分渠道ROI、获客成本、LTV/CAC比

常见错误：渠道增长归因为市场大盘；ROI计算忽略时间成本
动作词库：拓展、投放、优化、谈判、核算、置换、获客
```

### 社群运营

```
核心公式：社群运营 = 社群搭建 + 活跃维护 + 用户粘性 + 私域转化

考察维度：
1. 社群搭建力 — 从0到1搭建并规模化社群
   证据：社群数量与规模、入群漏斗、分层群组设计
2. 活跃维护力 — 通过内容和活动维持社群活跃
   证据：活跃率、发言率、话题互动、KOC培养
3. 用户粘性力 — 让用户对社群产生归属和依赖
   证据：留存率、复访率、自发传播、社群文化
4. 私域转化力 — 将社群关系转化为业务价值
   证据：群内成交额、转化率、裂变拉新

常见错误：用群人数代替活跃人数；转化数据无法归因
动作词库：搭建、运营、活跃、转化、裂变、维护、触达
```

### 数据运营

```
核心公式：数据运营 = 指标体系 + 分析洞察 + 策略建议 + 业务联动

考察维度：
1. 指标体系力 — 搭建和优化业务指标体系
   证据：指标字典、北极星指标定义、指标拆解
2. 分析洞察力 — 从数据中发现业务问题和机会
   证据：异常归因、趋势预测、用户分群分析
3. 策略建议力 — 将分析结论转化为可执行的运营建议
   证据：策略文档、建议被采纳率、效果追踪
4. 业务联动力 — 推动业务方基于数据做决策
   证据：数据驱动的业务决策案例、跨部门数据协作

常见错误：把取数/SQL写成数据分析；分析结论无业务action
动作词库：分析、归因、监控、预测、驱动、洞察、建议
```

### 生态运营

```
核心公式：生态运营 = 平台治理 + 资源整合 + 生态合作 + 规则设计

考察维度：
1. 平台治理力 — 制定和维护平台规则与秩序
   证据：平台规则、治理策略、纠纷处理机制
2. 资源整合力 — 整合平台内外资源推动生态发展
   证据：服务商生态、开发者社区、合作伙伴网络
3. 生态合作力 — 推动关键生态伙伴的深度合作
   证据：战略合作项目、联合方案、生态赋能
4. 规则设计力 — 设计激励和约束机制引导生态行为
   证据：激励政策、合规规则、利益分配机制

常见错误：平台自然增长归因为自己的生态运营；合作伙伴数量≠生态价值
动作词库：整合、治理、合作、建设、规则、激励、赋能
```

## 产品大类

### C 端产品经理

```
核心公式：C端产品 = 用户洞察 + 体验设计 + AB验证 + 数据驱动

考察维度：
1. 用户洞察力 — 理解海量用户的深层需求和行为模式
   证据：用户研究、行为分析、竞品调研、场景拆解
2. 体验设计力 — 设计简洁高效的C端交互和功能
   证据：原型/交互稿、设计规范、交互优化案例
3. AB验证力 — 用实验验证产品假设
   证据：AB测试设计、实验结果分析、灰度策略
4. 数据驱动力 — 用数据决策产品方向和优先级
   证据：核心指标定义、数据复盘、指标拆解

常见错误：功能罗列无用户价值；AB测试结果无统计显著性
动作词库：设计、验证、优化、洞察、迭代、上线、实验
```

### B 端产品经理

```
核心公式：B端产品 = 业务理解 + 效率设计 + 权限模型 + 客户落地

考察维度：
1. 业务理解力 — 深度理解企业客户的工作流程和痛点
   证据：客户调研、业务流程图、行业知识、需求文档
2. 效率设计力 — 设计提升企业效率的工具和工作流
   证据：效率指标变化、操作步骤简化、自动化方案
3. 权限模型力 — 设计多角色、多层级的权限和协作体系
   证据：RBAC设计、组织架构映射、审批流设计
4. 客户落地上力 — 推动产品在企业客户侧的交付和使用
   证据：POC管理、客户培训、上线支持、续约率

常见错误：功能列表无效率提升数据；企业需求写成个人需求
动作词库：设计、交付、配置、对接、效率、权限、落地
```

### 商业化产品经理

```
核心公式：商业化产品 = 变现设计 + 定价策略 + 广告系统 + 收入增长

考察维度：
1. 变现设计力 — 设计产品的付费模式和变现路径
   证据：付费SKU、订阅体系、增值服务、广告位设计
2. 定价策略力 — 通过数据和实验优化定价
   证据：价格测试、套餐设计、折扣模型、竞品对标
3. 广告系统力（B端商业化方向）— 设计广告投放和竞价系统
   证据：CPC/CPM/CPA模型、广告定向、竞价策略、填充率
4. 收入驱动力 — 产品变化对收入的直接影响
   证据：ARPU变化、付费率、广告收入、ROI

常见错误：收入增长归因大盘；广告指标（CTR/CVR）无行业对标
动作词库：设计、变现、定价、投放、增收、优化、实验
```

### AI 产品经理

```
核心公式：AI产品 = 模型理解 + 能力定义 + 体验设计 + 效果评估

考察维度：
1. 模型理解力 — 理解LLM/模型的能力边界、工作原理和成本
   证据：模型选型、Prompt设计、能力评估、成本核算
2. 能力定义力 — 定义 AI 产品的能力范围和产品路线图
```

```
   证据：策略调整归因、成本节约、收入增长归因
```

```
   证据：行业报告、竞品对标、市场研究、趋势分析

   证据：市场规模测算、盈利模型、unit economics

   证据：被采纳的建议、决策后的业务结果、决策周期缩短

常见错误：行业报告无公司级洞察；商业建议无落地可行性
```

```
   证据：战略落地追踪、长期价值创造、竞争地位变化

常见错误：宏大的行业分析无公司级结论；战略建议不可执行
动作词库：研究、规划、分析、论证、建议、评估、追踪
```
